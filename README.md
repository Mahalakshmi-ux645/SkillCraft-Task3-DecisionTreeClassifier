# SkillCraft-Task3-DecisionTreeClassifier
This project implements a Decision Tree Classifier using the UCI Bank Marketing Dataset to predict customer subscription to a term deposit. It includes data preprocessing, categorical encoding, model training, evaluation using accuracy, confusion matrix, classification report, and decision tree visualization with Python and Scikit-learn.
# SkillCraft Technology Internship - Task 03
## Decision Tree Classifier using Bank Marketing Dataset
### 📌 Project Overview
This project is part of the SkillCraft Technology Data Science Internship. The objective is to build a Decision Tree Classifier that predicts whether a customer will subscribe to a bank term deposit based on demographic and behavioral data from the UCI Bank Marketing Dataset.
## 🎯 Objective
Develop a machine learning classification model to predict customer subscription decisions using the Decision Tree algorithm.
## 📂 Dataset
- **Dataset:** Bank Marketing Dataset
- **Source:** UCI Machine Learning Repository
- **Target Variable:** `y` (Customer subscribed: Yes/No)
## 🛠️ Technologies Used
- Python
- Google Colab
- Pandas
- NumPy
- Matplotlib
- Seaborn
- Scikit-learn
## 📋 Project Workflow
1. Import required libraries.
2. Load the Bank Marketing dataset.
3. Explore and understand the dataset.
4. Check for missing values.
5. Encode categorical features using Label Encoding.
6. Split the dataset into training and testing sets.
7. Train a Decision Tree Classifier.
8. Predict customer subscriptions.
9. Evaluate the model using:
   - Accuracy Score
   - Confusion Matrix
   - Classification Report
10. Visualize the Decision Tree.
## 📊 Results
The Decision Tree Classifier was successfully trained and evaluated. The model predicts whether a customer is likely to subscribe to a term deposit based on the provided features. Performance was assessed using Accuracy Score, Confusion Matrix, and Classification Report.
